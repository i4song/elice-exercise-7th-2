# elice-exercise-7th-2

엘리스에서 진행하는 AI 트랙 5기에 사용되는 강의 자료입니다.

## 실습 방법

1. 해당 폴더로 이동합니다.

    ex) `problems/problem1`

    ```bash
    cd problems/propblem1
    ```

2. 패키지를 설치합니다.

    ```bash
    npm install
    ```

3. 풀이 작성 후 실행시켜 테스트합니다.

    ```bash
    npm start
    ```
